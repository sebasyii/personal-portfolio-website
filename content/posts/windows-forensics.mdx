## Windows Forensics Cheatsheet

> This cheatsheet is a collection of useful commands and artifacts location for Windows Forensics. By no means is this a complete list of commands, but it is a good starting point.
> I create this because I often find myself finding the same commands over and over again, and I want to keep them organized.
> I hope you find this useful, and if you have any suggestions for new commands, please let me know!

## Jump list
