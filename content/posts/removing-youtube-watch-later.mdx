The youtube Watch Later Playlist can pile up due to a lot of videos. Here is a quick code snippet you can paste on the Youtube Watch Later Playlist page to remove all of it.

Referenced from [here](https://gist.github.com/astamicu/eb351ce10451f1a51b71a1287d36880f).

> The selector might change so change it if it is not working.

```javascript
setInterval(function () {
  document
    .querySelector('#primary #menu #button button[aria-label="Action menu"]')
    .click();
  var things = document.evaluate(
    '//span[contains(text(),"Remove from")]',
    document,
    null,
    XPathResult.ORDERED_NODE_SNAPSHOT_TYPE,
    null
  );
  for (var i = 0; i < things.snapshotLength; i++) {
    things.snapshotItem(i).click();
  }
}, 1000);
```
