```javascript
setInterval(() => {
  document
    .querySelector('#primary #menu #button button[aria-label="Action menu"]')
    .click();

  const things = document.evaluate(
    '//span[contains(text(),"Remove from")]',
    document,
    null,
    XPathResult.ORDERED_NODE_SNAPSHOT_TYPE,
    null
  );

  for (let i = 0; i < things.snapshotLength; i++) {
    things.snapshotItem(i).click();
  }
}, 1000);
```

The Original code is referenced from [here](https://gist.github.com/astamicu/eb351ce10451f1a51b71a1287d36880f).

> The selector might change so change it if the element has changed.

## How to use the code

Visit the [Youtube Watch Later](https://www.youtube.com/playlist?list=WL) page.

Open the Developer Console with the shortcut, <Kbd>Cmd</Kbd> + <Kbd>Shift</Kbd> + <Kbd>J</Kbd>.

Paste the code in the Developer Console.

_Should the code not work, refresh the page and try again._
